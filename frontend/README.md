# CosmoView Frontend

This directory contains the React frontend for CosmoView.

Use the root [`README.md`](../README.md) for the full project setup, architecture summary, and deployment instructions.

Quick commands:

```bash
npm install
npm run dev
npm run check
npm run build
```

Required env file:

```env
VITE_API_BASE_URL=http://localhost:3001
```
